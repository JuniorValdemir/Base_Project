JavaScript

Uma linguagem de programação

Como surgiu

O JS ou JavaScript é uma linguagem de programação de alto-nível, criada no meio da década de 90, mais precisamente em 1996 pelo lendário programador Brendan Eich que, além de criar o JavaScript, foi também um dos fundadores da Mozilla Corporation.

O JavaScript, a princípio, foi criado com o objetivo principal de validar formulários HTML. Na verdade, o JavaScript foi concebido para rodar no servidor. No entanto, com o passar do tempo isso foi modificado e, hoje, o JavaScript é uma linguagem Client-side.

O JavaScript teve vários nomes. O primeiro foi Mocha, o segundo foi LiveScript, e quando a Netscape passou a ter suporte á tecnologia Java em seu navegador, mudou o nome para JavaScript, como um jogo de marketing, para popularizar o script.

"A linguagem de programação JavaScript nos permite executar instruções como resposta às ações do usuário."

Cusiosidades sobre JavaScript

  • Nulo é um objeto.
  • Não lê e retorna as cores em hexadecimal, mas em RGB.
  • NaN é um número.
  • Indefinido pode ser definido, ou seja, não é uma palavra reservada.
